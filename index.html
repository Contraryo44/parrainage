Questions fréquentes

Comment fonctionne le parrainage ?

Cliquez sur le lien ou saisissez le code au moment de votre inscription. Le bonus est automatiquement attribué à votre compte une fois les conditions remplies (premier dépôt, première transaction, etc.).

Ces codes sont-ils vraiment actifs en 2026 ?

Oui, tous les codes et liens présentés sur ce site sont vérifiés régulièrement. Si une offre expire, elle est retirée ou mise à jour dans les plus brefs délais.

Y a-t-il des conditions pour recevoir le bonus ?

Chaque établissement fixe ses propres conditions. Pour Gaz de Bordeaux, le filleul doit être un nouveau client et saisir le code lors de sa souscription. La prime est déduite sur la prochaine facture dans les 3 mois.

Ce site est-il rémunéré ?

Ce site est tenu par un particulier qui partage ses propres codes de parrainage. En utilisant ces liens, vous l'aidez à percevoir un bonus — sans aucun coût supplémentaire pour vous.

Puis-je utiliser plusieurs codes à la fois ?

Oui, chaque code est indépendant par établissement. Vous pouvez par exemple ouvrir un compte Fortuneo et souscrire chez Gaz de Bordeaux simultanément.
